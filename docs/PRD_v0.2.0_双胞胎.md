# PRD v0.2.0 — 双胞胎八字功能

## 1. 背景

双胞胎八字按传统排盘完全一样，但现实中命运不同。命理上，先出生者禀支中**本气**，后出生者禀支中**中余气**。v0.2.0 实现此差异化排盘。

## 2. 功能范围

### 2.1 输入区新增「双胞胎」选项

| 字段 | 选项 | 默认 |
|---|---|---|
| 双胞胎 | 无 / 兄姐(先) / 弟妹(后) | 无 |

- 选择「兄姐」→ 按现有逻辑（本气），不变
- 选择「弟妹」→ 日/时/命/身四支取中余气
- 选择「无」→ 现有单栏布局

### 2.2 双胞胎模式布局

选择「兄姐」或「弟妹」后，页面切换为**双栏对比布局**：

```
┌─ 输入区 ─────────────────────────────────┐
│  姓名/性别/日期/时间/双胞胎: 兄姐│弟妹     │
├──────────────┬──────────────┤
│    大宝       │     小宝      │
│  年 月 日 时  │  年 月 日 时  │
│  (本气)      │  (中余气)     │
├──────────────┤              │
│  命 身 胎    │  命 身 胎     │
├──────────────┼──────────────┤
│  大运+流年   │  大运+流年    │
└──────────────┴──────────────┘
```

- 年柱月柱两边相同
- 日柱、时柱、命宫、身宫——小宝侧用中余气
- 岁运版块移到四柱下方，各看各的

### 2.3 藏干替换规则

仅日支、时支、命宫支、身宫支生效。

| 支 | 本气(大宝) | 后出(小宝) |
|---|---|---|
| 子 | 癸 | 壬(劫气) |
| 丑 | 己 | 癸+辛(中+余) |
| 寅 | 甲 | 丙+戊(中+余) |
| 卯 | 乙 | 甲(劫气) |
| 辰 | 戊 | 乙+癸(中+余) |
| 巳 | 丙 | 庚+戊(中+余) |
| 午 | 丁 | 己(中气) |
| 未 | 己 | 丁+乙(中+余) |
| 申 | 庚 | 壬+戊(中+余) |
| 酉 | 辛 | 庚(劫气) |
| 戌 | 戊 | 辛+丁(中+余) |
| 亥 | 壬 | 甲(中气) |

> 独气支（子卯酉）取劫气（同五行异阴阳），多气支取中气+余气，双气支取中气。

### 2.4 连锁影响

支主星改变后，以下自动变化：
- 支的十神（`zhi_shi_shen`）
- 支的十二长生（`chang_sheng`）
- 藏干展示（`cang_gan`）

## 3. 技术方案

### 3.1 后端 `api/handler.rb`

- 新增常量 `ZHI_TWIN_MAIN`、`ZHI_TWIN_CANG`
- `pillar_info` 方法增加 `pillar_type` 和 `twin` 参数
- `/paipan` 端点接收 `twin` 参数（1=本气/默认，2=后出）
- 前端双胞胎模式下调用两次 API（twin=1 和 twin=2）

### 3.2 前端 `standalone.html`

- 输入区加双胞胎下拉
- 双胞胎模式切为两栏布局
- 两栏各自调用 `/paipan?twin=1` 和 `/paipan?twin=2`
- 底部版本号更新为 `从真版 v0.2.0`

### 3.3 面板 `panels/paipan/view.js`

- 输入区同步加双胞胎选项
