## 2. 节气计算（天文算法）

### 2.1 24 节气列表

```
小寒 大寒 立春 雨水 惊蛰 春分 清明 谷雨
立夏 小满 芒种 夏至 小暑 大暑 立秋 处暑
白露 秋分 寒露 霜降 立冬 小雪 大雪 冬至
```

索引：0-23，每个 15°，小寒=285°

### 2.2 太阳视黄经计算

基于 Meeus 低精度公式（误差约 1'）：

```
T = (JD - J2000) / 36525                    // 儒略世纪数

// 平近点角
M = 357.5291092 + 35999.0502909·T - 0.0001536·T²

// 平黄经
L₀ = 280.46645 + 36000.76983·T + 0.0003032·T²

// 中心差
C = (1.914602 - 0.004817·T - 0.000014·T²)·sin(M)
  + (0.019993 - 0.000101·T)·sin(2M)
  + 0.000289·sin(3M)

// 章动修正
Ω = 125.04 - 1934.136·T
L = L₀ + C - 0.00569 - 0.00478·sin(Ω)     // 真黄经 + 光行差 + 章动
```

其中 J2000 = 2451545.0（2000-01-01 12:00 UTC），所有角度用度为单位。

### 2.3 节气时刻计算

二分法搜索，目标黄经 = (n × 15 + 285)°，n=0..23。

初值用经验参数估算，然后在 ±1 天内二分法迭代 32 次（精度 ~0.1ms）。

返回值转换为北京时间（UTC+8）。

## 3. 年柱

### 规则

以**立春**为界。如果公历出生日期在立春之前，年柱使用前一年。

### 公式

年干支 index = (年份 - 4) mod 60

基准：1984 年 = 甲子年（index = 0）。

---

## 4. 月柱

### 4.1 月支

以节气月（非农历月）为准。正月为寅月（立春→惊蛰前），
十二月为丑月（小寒→立春前）。

### 4.2 月干 — 五虎遁

| 年干 | 正月（寅）月干 |
|------|-------------|
| 甲、己 | 丙         |
| 乙、庚 | 戊         |
| 丙、辛 | 庚         |
| 丁、壬 | 壬         |
| 戊、癸 | 甲         |

口诀：**甲己之年丙作首，乙庚之岁戊为头，丙辛必定寻庚起，丁壬壬位顺行流，若问戊癸何方发，甲寅之上好追求。**

月干依次顺推：正月月干 + (月份序号 - 1) mod 10。

---

## 5. 日柱

### 公式

以 1900-01-01 = 甲戌（index = 10）为基准。

```
diffDays = 出生日期(UTC) - 1900-01-01(UTC) 的天数差
index = (diffDays mod 60 + 10) mod 60
```

日干 = TG[index mod 10]，日支 = DZ[index mod 12]

---

## 6. 时柱

### 6.1 时支

| 时间范围 (地方时) | 时支 | 时辰序号 |
|------------------|------|---------|
| 23:00-00:59      | 子   | 0       |
| 01:00-02:59      | 丑   | 1       |
| ...              | ...  | ...     |
| 21:00-22:59      | 亥   | 11      |

公式：时支 index = floor((小时 + 1) / 2) mod 12

### 6.2 时干 — 五鼠遁

| 日干 | 子时时干 |
|------|---------|
| 甲、己 | 甲     |
| 乙、庚 | 丙     |
| 丙、辛 | 戊     |
| 丁、壬 | 庚     |
| 戊、癸 | 壬     |

口诀：**甲己还加甲，乙庚丙作初，丙辛从戊起，丁壬庚子居，戊癸何方发，壬子是真途。**

时干 = (五鼠遁[日干]的 index + 时支序号) mod 10

## 7. 命宫

### 7.1 地支序号

地支以寅=1 起序（寅1 卯2 辰3 巳4 午5 未6 申7 酉8 戌9 亥10 子11 丑12）。

### 7.2 公式

```
命宫地支数 = 14 - 月支数 - 时支数
若 ≤ 0 则 +12
若 > 12 则 -12

命宫天干 = 按五虎遁，以年干定正月月干，顺数至命宫地支所在月
```

---

## 8. 身宫

### 公式

```
身宫地支数 = (月支数 + 时辰序号) mod 12
时辰序号：子=1, 丑=2, ..., 亥=12
若结果为 0 则取 12

身宫天干 = 按五虎遁，以年干定正月月干，顺数至身宫地支所在月
```

---

## 9. 胎元

### 公式

```
胎元天干 = (月干 index + 1) mod 10
胎元地支 = (月支 index + 3) mod 12
```

即：月干顺推一位，月支顺推三位。

## 10. 大运 & 起运

### 10.1 顺逆判断

| 性别 | 年干阴阳 | 方向 |
|------|---------|------|
| 男   | 阳      | 顺   |
| 男   | 阴      | 逆   |
| 女   | 阳      | 逆   |
| 女   | 阴      | 顺   |

阳干：甲丙戊庚壬；阴干：乙丁己辛癸。

### 10.2 起运天数

顺排：从出生日到下一个节气的天数。
逆排：从上一个节气到出生日的天数。

### 10.3 起运年龄

```
起运总月数 = 起运天数 / 3 × 12
起运年龄 = floor(总月数 / 12) 岁 + (总月数 mod 12) 个月
```

即：3 天 = 1 岁起运。

### 10.4 大运排列

从月柱出发，顺排则依次顺推，逆排则依次逆推。每运 10 年，共排 10 步大运。

大运干支 index = (月干支 index ± n) mod 60（n=1..10，顺为+n，逆为-n）。

## 19. 真太阳时

### 19.1 均时差（Equation of Time）

Spencer 1971 公式：

```
B = (dayOfYear - 1) × 2π / 365
EoT = 229.18 × (0.000075
     + 0.001868·cos(B) - 0.032077·sin(B)
     - 0.014615·cos(2B) - 0.040849·sin(2B))
```

返回值：分钟（正值 = 真太阳超前于平太阳）。

### 19.2 真太阳时换算

```
经度偏移 = (当地经度 - 120) × 4 分钟
总偏移 = 经度偏移 + 均时差
真太阳时 = 北京时间 + 总偏移
```

处理跨日边界。
