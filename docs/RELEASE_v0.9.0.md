## 新增功能

### 🗂️ 档案管理（核心模块）
- **档案卡片列表**：美观卡片式展示，每张卡片显示小名、姓名、性别图标、八字四柱摘要、保存时间
- **搜索过滤**：输入即搜，模糊匹配小名和姓名，支持中文单字
- **展开排盘**：点击卡片展开完整八字排盘（四柱+三垣+宫位标签+大运+流年），手风琴模式
- **加载到表单**：一键回填表单并自动排盘
- **小名字段**：输入表单新增「小名」字段，选填不阻断保存

### 👶 预置数据
- **自在班16孩**：首次加载自动写入16条预置档案（六一、子旭、子阳、希希、东东、糯米、想想、NONO、扬扬、顶顶、KK、二哥、茂茂、成哥、笑笑、妍语）
- 预置档案含完整生日信息，八字首次展开时实时计算并缓存

### 💾 数据模型升级
- 存储键名升级为 `bz_archives_v2`（含完整八字结果：bazi/sanyuan/extras）
- 旧版 `bz_archives` 首次加载时自动迁移，数据不丢失
- 保存时同步存储排盘计算结果，展开无需重新计算
