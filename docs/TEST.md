# 测试用例

> 覆盖核心排盘算法。每次发版前必须全部通过。  
> 运行方式：浏览器打开 `tests/runner.html`，点击「运行全部测试」，全部 PASS 才允许发版。

## 运行方式

1. 打开 `/api/ext/bazi-paipan/tests/runner.html`（独立页面入口）
2. 或直接浏览器打开 `tests/runner.html`
3. 点击「▶ 运行全部测试」
4. 所有断言 **PASS** → 通过。有任何 **FAIL** → 禁止发版，先修 bug

## 测试设计原则

1. **纯函数测试**：所有算法函数（shiShen、changSheng 等）不依赖 DOM，直接调函数比结果
2. **全盘回归**：`paiPan()` 输出完整 chart 对象，逐字段断言
3. **不可变数据**：9 组八字来自学员档案，是黄金标准——预期值不能随意改

## 自动化方向（v0.4.0）

1. `runner.html` 已实现一键验证
2. 每次改算法后，打开 `runner.html` 跑全部
3. 全部 PASS 才允许 commit
4. 未来可在 CI 中用 headless Chrome 自动跑
